## Overview 📌
Dext System Agents is a pragmatic, spec-driven agent setup for OpenCode that emphasizes clean engineering practices, secure workflows, and consistent communication. It codifies persona, design principles, testing discipline, and memory management to deliver predictable, high-quality outcomes.

## Features ✨
- Specification- and test-first workflow aligned with project specs and backlog
- Clean, minimal design conventions with responsive, dark/light-ready UI guidance
- Secure operating rules with explicit guardrails for destructive actions
- Persistent memory via an Obsidian vault for project sessions and learnings
- Multi-agent configuration with tuned models and temperatures
- MCP integrations for Context7 and local Obsidian workflows

## Setup ⚙️
1. Install OpenCode and ensure it can read this repository.
2. Verify `opencode.json` is present at the project root.
3. Confirm MCP endpoints are reachable (Context7 remote, Obsidian local).

## Usage ▶️
- Use the orchestrator for spec-driven planning and phase handoffs.
- Follow the developer workflow to implement, test, and document changes.
- Keep communication clean and professional; Spanish for user-facing chat, English for technical artifacts.
